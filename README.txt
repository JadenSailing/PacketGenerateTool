目录说明：
配置Config.ini 里生成路径
\packet 消息的定义
\output 输出lua文件的相对路径，Config里可配置绝对路径
\src 工具源码
\python38 python3库

典型格式 参见"TestPacket.txt"
每个配置文件可以根据模块需求配置多条消息
但是由于项目设定，每条消息会单独生成一个lua文件
生成的路径可以自定义

相对宽松的格式要求：
{}用于消息或结构设置
""用于设置字符串默认值
//用于注释
除此之外不要使用其它符号，空格任意

支持特性
	*支持全数据类型及对应数组
	
	*支持自定义结构体，对应数组以及结构体嵌套处理等
	
	*支持消息头注释和属性的行尾注释 "//"
	
	*某些情况下需要分支解析消息，所以支持外层case条件判断
	case type value 
		xxxxx
	endcase
